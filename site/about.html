Описание проекта

VIDEOTEXT — модуль образовательной платформы Syllabica, предназначенный для создания и обработки видеотекстов на основе JSON-сценариев и инструментов визуализации.


Проект объединяет веб-интерфейс, систему рендеринга, механизмы обработки текстовых данных и средства автоматической генерации анимации.

Цель проекта

Разработка системы для создания, редактирования и визуализации видеотекстов с использованием современных веб-технологий и инструментов обработки данных.

Основные задачи

Frontend

Создание интерфейсов и страниц веб-приложения.

Backend

Интеграция Flask, очередей задач и API.

JSON

Работа со сценариями видеотекстов и данными.

GitHub
