Разработка системы VIDEOTEXT

VIDEOTEXT — это модуль платформы Syllabica, предназначенный для создания, обработки и визуализации видеотекстов с использованием Flask, JSON и Python.

В рамках проектной практики был разработан статический сайт проекта, оформлена документация, освоены Git и Markdown, а также создан репозиторий GitHub.
