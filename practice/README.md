# 1. Введение

В рамках проектной (учебной) практики проводилась работа над проектом VIDEOTEXT — модулем платформы Syllabica, предназначенным для создания, обработки и визуализации видеотекстов.

Во время практики были изучены принципы работы Flask-приложений, организация маршрутизации, работа с JSON-данными, а также основы проектирования пользовательских интерфейсов и публикации статических сайтов с использованием GitHub Pages.

Практика также включала работу с Git, Markdown-документацией и разработку многостраничного HTML/CSS-сайта проекта.
