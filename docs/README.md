# Документация проекта

В данной папке размещены материалы, связанные с документацией проекта VIDEOTEXT и выполнением проектной (учебной) практики.

## Содержание

- описание проекта;
- материалы по разработке;
- документация в формате Markdown;
- дополнительные сведения по проектной практике.

## Используемые технологии

- HTML5
- CSS3
- Python
- Flask
- GitHub
- Markdown

## Назначение документации

Документация предназначена для описания структуры проекта, этапов разработки и используемых технологий.
