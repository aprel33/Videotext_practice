# ВВЕДЕНИЕ

Целью проектной практики являлась разработка статического веб-сайта проекта VIDEOTEXT, оформление документации в Markdown и освоение инструментов Git и GitHub.

Проект VIDEOTEXT связан с платформой Syllabica и предназначен для создания, обработки и визуализации видеотекстов. В рамках практики необходимо было представить основную проектную работу в виде сайта, подготовить репозиторий, оформить документацию и опубликовать результат через GitHub Pages.

Во время выполнения практики были изучены основы HTML и CSS, структура многостраничного сайта, принципы оформления Markdown-документов, работа с системой контроля версий Git, а также публикация статического сайта в открытом доступе.

## 4.3. Работа с проектом VIDEOTEXT

Проект VIDEOTEXT является частью платформы Syllabica и предназначен для создания и визуализации видеотекстов.

В ходе изучения проектной работы были рассмотрены следующие элементы:

- структура Flask-приложения;
- работа с JSON-сценариями;
- организация frontend-страниц;
- отображение данных в HTML-интерфейсе;
- интеграция отдельных модулей проекта;
- асинхронная обработка задач рендеринга.

Особое внимание было уделено модулю `video_queue.py`, который используется для организации очереди задач рендеринга. Такой подход позволяет не блокировать основной поток Flask-приложения и выполнять ресурсоёмкие задачи в фоновом режиме.

Также была изучена работа с JSON-данными, которые используются для хранения и передачи сценариев видеотекстов.
