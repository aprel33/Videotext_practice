# Отчёты по проектной практике

В данной папке размещены отчёты по проектной практике в бинарных форматах.

## Содержание

- [report.docx](report.docx) — отчёт по проектной практике в формате Microsoft Word.
- [report.pdf](report.pdf) — PDF-версия отчёта.

## Описание

Отчёт содержит:
- описание проекта VIDEOTEXT;
- цели и задачи проектной практики;
- описание выполненных работ;
- достигнутые результаты;
- выводы по итогам практики.

## Дополнительные материалы

Структура отчёта была оформлена в соответствии с шаблоном проектной практики Московского Политеха.
